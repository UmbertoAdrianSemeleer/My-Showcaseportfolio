Project Overview: The Beach

Project Overview

My second project in game design was the "Dual Project," where we were assigned to build any game we wanted. My partner and I decided to create a 2D platformer game because we wanted to learn about 2D game development.

We had only four weeks to complete the project. Due to time constraints, we focused on implementing the core mechanics and decided to make it a 2.5D game along the way.

The concept we developed was "The Beach," a 2.5D shooting platformer game.
